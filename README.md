# Dr. Escriba IA · Edição Internato MGF

Aplicação **single-file** para apoio documental ao **internato de Medicina
Geral e Familiar** em Portugal.

## Componentes

- **Modo SOAP** (igual ao [drescribia](https://github.com/proteinaludica/drescribia)
  para especialistas): notas estruturadas para Urgência, Visita Domiciliária,
  Consulta e Internamento.
- **Modo Internato** (exclusivo): Logbook OM, Avaliações, Diário, Journal Club.

## Princípios

- **Privacidade**: tudo fica no browser. Zero servidores, zero analytics.
- **Não-clínico**: assistente documental. A decisão clínica é do médico.
- **Single-file**: `index.html` é a aplicação inteira. Funciona em
  `file://`, USB, sem build.
- **PT-PT**: terminologia clínica portuguesa, rubricas OM oficiais.

## Uso

Abrir `index.html` num browser moderno (Chrome, Edge, Safari, Firefox).
Toggle SOAP/Internato no canto superior direito.
